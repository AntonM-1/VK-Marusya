# Маруся (Cinemaguide)

Учебный проект — одностраничное приложение-каталог фильмов: подборка топ-фильмов, поиск, страницы жанров, карточка фильма с трейлером, авторизация и избранное.

## Стек

- React 19 + TypeScript
- Vite
- React Router
- Redux Toolkit + React Redux — хранение избранного
- Sass (модули)
- Vitest + Testing Library — тесты
- ESLint

## Возможности

- Главная страница со случайным фильмом в баннере и списком топ-фильмов
- Поиск фильмов по названию
- Список жанров и фильмы по жанру
- Страница фильма с трейлером (модалка) и подробным описанием
- Регистрация / вход / выход (сессия на cookie, бэкенд `cinemaguide.skillbox.cc`)
- Избранное: добавление и удаление фильмов, личный кабинет со списком избранного

## Структура

```
src/
  api/          запросы к бэкенду (auth, favorites, genres, movies)
  components/   переиспользуемые компоненты
  context/      React-контексты (авторизация, уведомления)
  hooks/        кастомные хуки
  pages/        страницы приложения
  store/        Redux store и слайсы (favorites)
  types/        общие TypeScript-типы
  utils/        вспомогательные функции
```

## Запуск

```bash
npm install
npm run dev
```

Приложение поднимется на `http://localhost:5173` и обращается к публичному учебному бэкенду `https://cinemaguide.skillbox.cc`.

## Скрипты

| Команда | Назначение |
| --- | --- |
| `npm run dev` | запуск dev-сервера с HMR |
| `npm run build` | проверка типов и продакшен-сборка |
| `npm run preview` | локальный просмотр собранного билда |
| `npm run lint` | проверка ESLint |
| `npm test` | однократный прогон тестов |
| `npm run test:watch` | тесты в watch-режиме |
